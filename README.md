# smdata
숙명여자대학교의 비교과 프로그램 추천 시스템입니다. (제4회 SM-data 분석 공모전 대상 수상)

학생의 비교과 활동 이력과 프로그램 태그 데이터를 활용하여  
콘텐츠 유사도 기반으로 향후 수강에 적합한 프로그램을 추천합니다.

본 프로젝트는 **TF-IDF + Cosine Similarity**를 활용한  
콘텐츠 기반 추천(Content-based Recommendation) 구조를 따릅니다.

---

## 📌 프로젝트 개요

이 시스템은 학생이 **과거에 수강한 프로그램의 내용적 특성**을 학습하여  
향후 개설 예정인 프로그램 중 **가장 유사도가 높은 프로그램**을 추천합니다.

주요 목표는 다음과 같습니다.

- 학생 이력 기반 개인화 추천
- 최근 수강 이력에 더 큰 가중치를 부여
- 이미 수강한 프로그램 자동 제외
- 추천 사유를 함께 제공하는 설명 가능한 추천

---

## 🧠 추천 로직 요약

### 1. 텍스트 전처리
- 프로그램 태그를 문자열로 변환
- 결측치 및 불필요한 값 제거
- 소문자 변환 및 공백 정규화

### 2. TF-IDF 벡터화
- 문자 단위(character n-gram) 기반 TF-IDF 적용
- n-gram 범위: 2~4
- 각 프로그램을 벡터로 표현

### 3. 학생 벡터 생성
- 학생이 수강한 프로그램 벡터들의 가중 평균
- 가중치 구성 요소:
  - 이수 여부
  - 최근성(Recency Weight, 시간 감쇠 함수)

### 4. 후보 프로그램 필터링
- 기준일 이후 개설 프로그램만 포함
- 이미 수강한 프로그램 제외
- 학년별 부적합 프로그램 제외
  - 예: 2~4학년 → ‘신입생’, ‘새내기’ 태그 프로그램 제외
- 선택적 필터 적용 가능:
  - 프로그램 기간 (일회성 / 단기 / 장기)
  - 프로그램 유형
  - 핵심역량(core)

### 5. 추천 결과 생성
- Cosine Similarity 기준 상위 프로그램 선택
- 과거 수강 프로그램과 가장 유사한 항목을 기반으로 추천 사유 생성

---

## 📂 데이터 구성 (주요 컬럼)

- student_id : 학생 ID
- program_id : 프로그램 ID
- program_name : 프로그램명
- tag : 프로그램 태그
- start / end : 프로그램 시작·종료일
- completed : 이수 여부
- type : 프로그램 유형
- core_1 ~ core_8 : 핵심역량 지표
